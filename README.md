# 🟢 Modern Portfolio - Matrix & macOS Edition

A high-performance, immersive developer portfolio built with **React**, **Vite**, and **Tailwind CSS**. This project features a unique "Matrix-style" visual identity combined with an interactive macOS-inspired code editor interface.

[![GitHub Repo](https://img.shields.io/badge/GitHub-Portfolio-00ff41?style=for-the-badge&logo=github)](https://github.com/Leozin2412/portfolio)

## 🚀 Features

- **Matrix Visual Identity:** Deep black backgrounds with neon green accents for a high-tech terminal vibe.
- **macOS User Experience:** 
  - Custom macOS-style cursors (arrow and hand pointer).
  - Mac window frame with interactive controls.
  - Smooth-scrolling single-page navigation.
- **Bilingual Support (i18n):** Real-time toggle between **English** and **Portuguese** via a flag switch in the header.
- **Terminal Boot Animation:** Immersive startup sequence simulating a system BIOS and kernel initialization.
- **Responsive Design:** Fully optimized for desktop and mobile devices.
- **Professional Layout:** 
  - **Home:** Hero section with tech stack icons.
  - **About:** Personal bio and educational background.
  - **Timeline:** Chronological professional experience.
  - **Contact:** Integrated email trigger, social links, and CV download.

## 🛠️ Tech Stack

- **Framework:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment Ready:** Optimized for platforms like Vercel or Netlify.

## 📁 Project Structure

```text
portfolio/
├── src/
│   ├── components/       # UI Components (MacWindow, Sidebar, etc.)
│   ├── context/          # Language/State Context
│   ├── locales/          # Translation dictionaries (EN/PT)
│   ├── sections/         # Page sections (Home, About, Timeline, etc.)
│   ├── App.jsx           # Main application logic & scroll handling
│   └── index.css         # Global styles & custom cursors
├── public/               # Static assets (Resume, Profile picture)
└── package.json          # Dependencies & Scripts
```

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Leozin2412/portfolio.git
   cd portfolio/portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
